# NearBite Admin Console

Production-ready admin console for the NearBite food delivery platform built with Angular 21.

## Quick Start

```bash
npm install
npm start
```

Login at http://localhost:4200/login with **admin@nearbite.com** (any password)

## Features

✅ Real-time Dashboard with platform metrics
✅ Role-based Access Control (6 roles)
✅ Restaurant, Order, Customer, Rider Management
✅ Financial Operations (Settlements, Refunds)
✅ Support Ticket System
✅ Content & Campaign Management
✅ Dark theme optimized for operations teams

## Tech Stack

- Angular 21 (Standalone Components)
- TypeScript 5.9
- SCSS with NearBite brand theme
- RxJS for state management
- Lazy-loaded routes
